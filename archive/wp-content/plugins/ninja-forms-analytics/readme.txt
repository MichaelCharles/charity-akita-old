Ninja Forms Analytics makes it easy for you to retrieve user traffic data such as IP Address, User Agent, Referrer URL and Mobile Device detection. 

== Description ==
Ninja Forms is the ultimate FREE form creation solution for WordPress. Build WordPress Forms within minutes using Ninja Forms powerful forms creation page. It allows you to easily and quickly design complex forms through a drag and drop interface and absolutely no code. 

With Ninja Forms Analytics, you can retrieve user traffic data such as IP Address, User Agent, Referrer URL and Mobile Device detection in every form submission by adding those hidden fields when creating new form.

Here are just a few of the things you will find in Ninja Forms Analytics:

* Custom fields at 'Template Fields' section:
1. User IP
2. User Agent
3. Referrer URL
4. Mobile Device

* Manage, Edit, and Export form user submissions with additional custom fields above.

= IMPORTANT NOTES =
*   In order to translate this plugin into your own language, do use POEdit or plugin and open 'ninja-forms-analytics.pot' file and save the file as ninja-forms-analytics-[language code].po, then put that into languages folder for this plugin. Or you can use plugin 'Codestyling Localization' and go to Tools > Localization.


== Installation ==

1. Upload entire folder 'ninja-forms-analytics' to the '/wp-content/plugins/'' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You will see new custom fields when add/edit form.
